# Omarchy-Stuff

Подборка моих патчей, настроек и небольших проектов для [Omarchy](https://omarchy.org).

Я давно и постоянно пользуюсь Omarchy как основной системой, поэтому складываю сюда не абстрактные заготовки, а вещи, которые реально использую сам. Это репозиторий с практической пользой: если что-то оказалось удобным в повседневной работе у меня, возможно, пригодится и вам.

## Что это за репозиторий

Здесь лежат доработки вокруг Omarchy, которые закрывают конкретные неудобства или добавляют недостающие мне возможности.

Основной принцип простой:

- в репозиторий попадает только то, чем я действительно пользуюсь сам
- решения должны быть воспроизводимыми и понятными
- каждая вещь должна давать реальную практическую пользу, а не просто существовать "на всякий случай"

## Что уже есть

### [omarchy-walker-layout-search](./omarchy-walker-layout-search/)

Двунаправленный поиск приложений EN↔RU для Walker в Omarchy.

Если вы случайно набрали запрос в неправильной раскладке, поиск все равно работает:

- `firefox` находится и по `ашкуащч`
- русские названия находятся и при вводе с английской раскладки

Проект патчит `desktopapplications` в Elephant, добавляет поддержку конвертации физической раскладки клавиатуры и собирает совместимый бинарник вместе с плагинами.

Подробнее: [README проекта](./omarchy-walker-layout-search/README.md)

## Для кого это может быть полезно

Этот репозиторий может пригодиться, если вы:

- тоже используете Omarchy каждый день
- любите небольшие, прикладные улучшения без лишней сложности
- хотите брать готовые решения и адаптировать их под себя
- ищете реальные примеры кастомизации, которые уже были использованы в живой системе

## Как пользоваться репозиторием

Каждый проект внутри репозитория живет в своей отдельной папке и обычно содержит:

- собственный `README.md`
- инструкции по установке
- примеры конфигурации
- скрипты для сборки, проверки или интеграции

Если вас интересует конкретная фича, просто открывайте соответствующую директорию из списка выше.
